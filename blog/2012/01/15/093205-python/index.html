2.1 行结构和缩进

  • 每条语句以换行符结束。可用续航符()分行。 
    a = math.cos(3 * (x - n) + \
  math.sin(3 * (y - n))
  • 包含在圆括号，方括号，大括号，三引号的程序部分可以跨多行。
  • 缩进用于表示不同的代码块，应用空格缩经，且整个代码的缩进必须保持一致。 
    if a:
    statement1    # 缩进一致，正确
    statement2
else:
    statement3
      statement4  # 缩进不一致，错误
  • 函数体，条件语句，循环或类较短，可放在一行。 
    if a: statement1
else: statement2
  • 表示一个空的函数体或代码块。 
    if a:
    pass
else:
    statements
  • 分号(;)可用于分割放在一行的多条语句。
  • \#表示次行的内容是注释。
