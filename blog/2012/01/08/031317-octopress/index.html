Octopress 像黑客一样写博客

Jan 8th, 2012

Octopress 简介

Octopress is a framework designed by Brandon Mathis for Jekyll, the blog aware static site generator powering Github Pages.

优点

用文件方式储存数据，无需数据库 以静态方式发布，可直接挂在github等地方 用markdown格式写博客 可以轻松的配置和发布

安装

$ git clone git://github.com/imathis/octopress.git octopress
$ mv octopress blog
$ cd blog
$ sudo gem install rake --version=0.9.2
$ sudo gem install bundler
$ bundler install
$ rake install

GitHub

$ rake setup_github_pages
>git@github.com:zhangchuan/zhangchuan.github.com.git

常用命令

添加文章
$ rake new_post["title"]
$ rake generate       #发布文件到public目录
$ rake watch          #监控source和sass目录的变动
$ rake preview        #启动服务器并监控变动，通过http://localhost:4000预览
发布
$ rake generate
$ rake deploy
保存源代码
$ git add .
$ git commit -m 'blog'
$ git push origin source

自定义样式

$ vi sass/custom/_styles.scss
// 博客标题
body > header h1 { font-size: 1.2em; }
body > header h2 { font-size: 0.8em; }
// 文章标题
.entry-title { font-size: 1.2em; }
.entry-content { font-size: 1.0em; }

article {
    font-size: 0.8em;
    line-height: 1.2em;
}
// 边栏
aside { font-size: 0.8em; }
// 时间
time { 
    font-size: 0.8em;
}

h1 { font-size: 1.2em; }
h2 { font-size: 1.1em; }
h3 { font-size: 1.0em; }
h4 { font-size: .9em; }
h5 { font-size: .8em; }
code { font-size: 0.8em; }
